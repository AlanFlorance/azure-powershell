# New-AzSynapseSqlPoolSensitivityLabel

## SYNOPSIS
Creates or updates the sensitivity label of a given column in a Sql pool

## SYNTAX

```
New-AzSynapseSqlPoolSensitivityLabel -ColumnName <String> -ResourceGroupName <String> -SchemaName <String>
 -SqlPoolName <String> -TableName <String> -WorkspaceName <String> [-SubscriptionId <String>]
 [-InformationType <String>] [-InformationTypeId <String>] [-LabelId <String>] [-LabelName <String>]
 [-Rank <SensitivityLabelRank>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Creates or updates the sensitivity label of a given column in a Sql pool
