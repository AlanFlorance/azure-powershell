# Get-AzSynapseSqlPoolTableColumn

## SYNOPSIS

```
 -TableName <String> -WorkspaceName <String> [-SubscriptionId <String[]>] [-Filter <String>]
 [-DefaultProfile <PSObject>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
