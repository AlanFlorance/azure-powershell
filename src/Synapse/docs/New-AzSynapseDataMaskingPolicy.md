# New-AzSynapseDataMaskingPolicy

## SYNOPSIS
Creates or updates a Sql pool data masking policy

## SYNTAX

```
New-AzSynapseDataMaskingPolicy -ResourceGroupName <String> -SqlPoolName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-DataMaskingState <DataMaskingState>] [-ExemptPrincipal <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates or updates a Sql pool data masking policy
