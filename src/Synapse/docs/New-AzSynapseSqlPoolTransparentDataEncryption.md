# New-AzSynapseSqlPoolTransparentDataEncryption

## SYNOPSIS
Creates or updates a Sql pool's transparent data encryption configuration.

## SYNTAX

```
New-AzSynapseSqlPoolTransparentDataEncryption -ResourceGroupName <String> -SqlPoolName <String>
 -WorkspaceName <String> [-SubscriptionId <String>] [-Status <TransparentDataEncryptionStatus>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates or updates a Sql pool's transparent data encryption configuration.
